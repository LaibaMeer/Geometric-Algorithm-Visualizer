Line Intersection

Choose from below algorithms to get started

Polygon Approach


Break each line into line segments, forming a polygon.

Find the intersection point within the polygon.

Algorithms like the even-odd rule or winding number algorithm, can be used to determine if a point is inside the polygon.

Time Complexity:O(N)

Determinant Approach


Approach :two lines in a 2D space intersect if and only if their determinants are not equal, and the lines are not parallel.

Given two lines with equation a2x + b2y =c1 and a2x + b2y =c2

Calculate determinant with the help of coefficient matrix : D = a1b2 - a2b1

If D != 0, Lines intersects at a point

To calculate point of intersection, use Cramer's Rule

Time Complexity:O(1)

Parametric Approach


In 2D, Solves a system of linear equations in constant time.

As it moves to higher dimensions (3D, 4D, etc.),the time complexity can become linear in the number of dimensions.

Time Complexity: O(1) for 2D & O(n) for 3D
